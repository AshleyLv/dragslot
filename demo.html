How to use it:

1. Including dragslot into your project.


<link rel="stylesheet" type="text/css" href="css/dragslot.css">
<script type="text/javascript" src="js/dragslot.js"></script>

2. Html code


<div class="slot">
 <ul class="slot-list">
  <li class="slot-item">
   <div class="slot-handler">
	drag item 1
   </div>
  </li>
  <li class="slot-item">
   <div class="slot-handler">
	drag item 2
   </div>
   </li>
 </ul>
</div>

3. init dragslot


$('#dragslot').dragslot({
  slotItemClass : 'slot-item',
  placeholderClass : 'placeholder',
  dragItemClass : 'drag-item',
  slotListClass : 'slot-list',
  slotHandlerClass : 'slot-handler',
  emptySlotClass : 'empty-slot',
  slotClass : 'slot',
  slotItem : 'li',
  slotList : 'ul',
  dropCallback: function(el){
    alert('dragItem : ' + el.dragItemId);
  }
});

4. dragslot arguments

slotItemClass
css class of slot items, the default value is 'slot-item'
placeholderClass
css class of slot item placeholder, the default value is 'placeholder'
dragItemClass
css class of slot item when it is moving, the default value is 'drag-item'
slotListClass
css class of slot list, the default value is 'slot-list'
slotHandlerClass
css class of the wrapper element of slot item, the default value is 'slot-handler'
emptySlotClass
css class of the empty slot, the default value is 'empty-slot'
slotClass
scc class of slot, the default value is 'slot'
slotItem
the html element of slot item, the default value is 'li'
slotList
the html element of slot list, the default value is 'ul'
dropCallback
callback function when the item is dropped
